A beginner-friendly Calculator Program in C that performs basic arithmetic operations using a switch-case statement.

Features
Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)
Division-by-zero validation
Simple command-line interface
Concepts Used
C Programming
Switch Case Statements
Conditional Logic
Variables and Data Types
User Input/Output Functions (scanf, printf)

This project is designed to help beginners understand the implementation of arithmetic operations and decision-making structures in C.
